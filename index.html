Connect tool

Press the spacebar to enable connect tool. Then, select two objects to connect. You can also delete an existing connection by doing the same thing, holding spacebar and selecting two objects that are already connected.

Version 1.1.0

7/13/2026 This update introduces refinements to the UI. A sidebar has been dedicated to the various modes the user may switch between: drag, select, and connect. Drag mode allows users to drag the blocks around. Select mode allows users to select text. Connect mode activates the connect tool. The favorites block list has been moved to a smaller menu in the sidebar. A new feature also allows users to save the favorites list. New users will also find the tutorial popup when they open the app for the first time.

Patch 1.1.1

7/13/2026 This patch fixes some backend workflows. It also prevents a ghost image from being created when dragging. Fix tutorial popup showing even when the autosave file is present.

Settings

Simrate

Set the simulation rate, measured in milliseconds. This is the amount of time between the updating of connections and block logics. Lower values may require more processing resources.Default: 500ms

ms

Autosave

Choose whether the program should automatically locally save the sandbox upon tab focus loss. Default: true

Save frames

Choose whether the program should locally save the 10 most recent frames for debug. Default: true

Show frame count

Choose whether you'd like to have the frame count shown in the debug tab. Default: false

Auto connect

Choose whether the connector tool should automatically create a connection if there are two selected elements that can be connected. This is helpful if you use the side toolbar rather than the keyboard shortcut. Default: true
