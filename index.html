Value: 0 Represents a binary 0
Value: 1 Represents a binary 1
Gate: NOT Inverts the input value
Gate: AND Outputs true if both inputs are true
Gate: OR Outputs true if at least one input is true
Gate: XOR Outputs true if one input is true and the other is false.
Special: Switch Can toggle between 0 and 1
Special: Lamp Lights up if the input is 1
